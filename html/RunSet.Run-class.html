make_run_from_file_step1(self, line)

source code 

Get run data from line.

This function is used to read in a file with pre-calculated run data written by 'get_normalisation'. This way, the normalisation can be done up to that point, the output file edited, and then read back in to continue with the modified runs. Format: Run Ein Eout Ecm +/- pressure FC4before +/- FC4after +/- tarTrans +/- sepTrans +/- HI_lt +/- CSF +/- SB0tot +/- SB0start +/- SB0end +/- Rstart +/- Rend +/- FC4/SB0start +/- FC4/SB0end +/- SB1tot +/- SB1start +/- SB1end +/- R1start +/- R1end +/- FC4/SB1start +/- FC4/SB1end +/-
